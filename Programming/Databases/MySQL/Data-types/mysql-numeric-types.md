## Описание

В MySQL можно найти все стандартные числовые типы SQL, включая точные числовые типы данных, а также приблизительные числовые типы данных, включая целые числа, с фиксированной запятой и с плавающей запятой.

| **Числовой тип** | **Описание** |
| ---- | ---- |
| `TINYINT` | Очень маленькое целое число |
| `SMALLINT` | Небольшое целое число |
| `MEDIUMINT` | Среднее целое число |
| `INT` | Стандартное целое число |
| `BIGINT` | Большое целое число |
| `DECIMAL` | Число с фиксированной точкой |
| `FLOAT` | Число одинарной точности с плавающей запятой |
| `DOUBLE` | Число двойной точности с плавающей запятой |
| `BIT` | Битовое поле |
